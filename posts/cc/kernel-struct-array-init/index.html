linux内核的结构体数组初始化方式

by YangYang
in c/c++ on 2017-07-06 | tags: 内核

话不多说，一看就懂：

typedef struct proto_id_str {
    int     id;
    char    str[4];
} proto_id_str_st;


proto_id_str_st transport_info[2] = {
        [0] = {
            .id     = 6,
            .str    = "TCP"
        },
        [1] = {
            .id     = 17,
            .str    = "UDP"
        }
    };

是不是感觉很干净，很整洁，很清晰。还是要多学学内核编码。
