编译安装python mssql模块

by YangYang
in python on 2017-12-21 | tags: pymssql

主要的步骤就是下面几个，网上下载相应的包，按以下流程走一遍，就ok了。博主亲测。

tar -xvzf freetds-0.82.tar.gz
cd freetds-0.82
./configure --prefix=/usr/local/freetds --with-tdsver=8.0 --enable-msdblib --enable-dbmfix --with-gnu-ld --enable-shared --enable-static
make && make install

tar -xvzf setuptools-38.2.4.zip
cd setuptools-38.2.4
python setup.py install

tar -xvzf setuptools-git-1.2.tar.gz
cd setuptools-git-1.2
python setup.py install

tar -xvzf pymssql-2.1.3.tar.gz
cd pymssql-2.1.3
python setup.py install

attention：freetds的库路径，要加到/etc/ld.so.conf中，否则会报找不到库的错误。
