安装php-redis扩展

by YangYang
in linux on 2017-02-18 | tags: php-redis 
1. 下载扩展包

wget https://github.com/nicolasff/phpredis/archive/master.zip

2. 解压安装redis扩展
unzip master.zip
cd phpredis-master
phpize
./configure –with-php-config=/usr/bin/php-config
make
make install

3. 配置PHP的配置文件,添加redis扩展
vim /etc/php.ini
extension=”redis.so”
